# Home Assistant

All Home Assistant related files

## Setup MQTT

- Open and login into Home Assistant
- Go to `Settings`
- Select `Devices and services`
- Click `Add Integration` on the lower right button to open the `Set up a new integration` dialog
- Select `MQTT`
- Enter the following parameters:
  - Broker: `mosquitto`
  - Port: `1883`
  - Username: `homeassistant`
  - Password: `<homeassistant password entered during the mosquitto setup>
`
